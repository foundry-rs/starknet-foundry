# `snforge clean`

Remove files generated by `snforge`.

## `all`

Clean all generated directories

## `cache`

Clean the cache directory.

> ℹ️ **Info**
>
> The default cache directory is `.snfoundry_cache`, relative to the workspace root.
> You can override it with the `SNFOUNDRY_CACHE` environment variable, which must be an absolute path.

## `trace`

Clean the `snfoundry_trace` directory.

## `coverage`

Clean the `coverage` directory.

## `profile`

Clean the `profile` directory.

## `-h`, `--help`

Print help.
