# `tx-status`

Get the status of a transaction

## `<TRANSACTION_HASH>`

Required.

Hash of the transaction

## `--url, -u <RPC_URL>`
Optional.

Starknet RPC node url address.

Overrides url from `snfoundry.toml`.

## `--network <NETWORK>`
Optional.

Use predefined network with public provider

Possible values: `mainnet`, `sepolia`.
