# `selector`
Calculate entrypoint selector from function name.

## `<NAME>`
Required.

The function name (e.g. `transfer`, `balance_of`). Do not include parentheses or parameter types.

## Example

```shell
$ sncast utils selector transfer
```

<details>
<summary>Output:</summary>

```shell
Selector: 0x0[..]
```
</details>
