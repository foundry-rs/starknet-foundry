# `ledger app-version`

Get the Starknet app version from the connected Ledger device.

No arguments are required. Make sure the Starknet app is open on the device before running.

## Example

<!-- { "requires_ledger": true } -->
```shell
$ sncast ledger app-version
```

<details>
<summary>Output:</summary>

```shell
App Version: 2.3.4
```
</details>
