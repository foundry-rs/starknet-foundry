# `ledger`

Interact with a Ledger hardware wallet.

See [Ledger Hardware Wallet](../../../starknet/ledger.md) for a full guide.

## Subcommands

* [app-version](./app-version.md) — Get the Starknet app version from the device
* [get-public-key](./get-public-key.md) — Read a public key from the device
* [sign-hash](./sign-hash.md) — Blind-sign a raw hash with the device
